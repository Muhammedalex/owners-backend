# Fonts Directory

## Tajawal Font

To use the Tajawal font in PDF invoices, please download and place the following font files in this directory:

1. **Tajawal-Regular.ttf** - Regular weight
2. **Tajawal-Bold.ttf** - Bold weight

### Download Links:
- Google Fonts: https://fonts.google.com/specimen/Tajawal
- Direct download: https://github.com/google/fonts/tree/main/ofl/tajawal

### Installation:
1. Download the font files from Google Fonts
2. Extract the `.ttf` files
3. Place `Tajawal-Regular.ttf` and `Tajawal-Bold.ttf` in this directory (`public/fonts/`)

### Note:
If the Tajawal font files are not found, the system will automatically use DejaVu Sans which has built-in Arabic support.
